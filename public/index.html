Sistema digital
¿Cómo funciona SIDEC Digital?

El sistema centraliza todos los certificados de calibración en una base de datos estructurada, facilitando la búsqueda, consulta y trazabilidad histórica desde 2015.

📂
Importación masiva
Los archivos XLSX, PDF y DOCX se importan desde carpetas organizadas por empleado y año.
🗄️
Base de datos PostgreSQL
Todos los campos del certificado se estructuran y almacenan con índices para búsqueda rápida.
🔍
Búsqueda avanzada
Filtra por cliente, magnitud, marca, No. serie, año, rango de fechas y más campos simultáneamente.
📋
Detalle completo
Consulta todos los datos del instrumento, condiciones ambientales, fechas y personal firmante.
✏️
Edición controlada
Los usuarios con rol Calidad o Admin pueden editar certificados con control de acceso por roles.
👥
Gestión de accesos
Roles diferenciados: Certificaciones, Reportes, Calidad y Admin con permisos granulares.
